# Poker-Hand-Sorter

A poker hand consists of a combination of five playing cards, ranked in the following ascending order (lowest to highest):

| Rank  | Combination      | 
| ----- |:----------------:| 
|   1   |   High card      |
|   2   |   Pair           |
|   3   |   Two Pairs      |
|   4   | Three of a kind  |
|   5   |   Straight       |
|   6   |   Flush          |
|   7   |   Full house     |
|   8   |   Four of a kind |
|   9   |   Straight flush |
|  10   |   Royal Flush    |


To run the program execute:

$ cat poker-hands.txt | python main.py
